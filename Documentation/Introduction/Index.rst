Introduction
------------

PayPal Subcription (paypal_subscription) is an extension for cart and cart_products to provide recuring transactions .

Focus
=====

The focus is to enable payment subscriptions with PayPal
To do so a cart_product can be configured as a PayPal product and plan over the api
Edit of Plan and product over the api is possible but not planned, yet.
The cart has to be configured as a single product buy method
